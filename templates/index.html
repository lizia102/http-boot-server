PXE/UEFI 启动说明

客户端通过网络启动时，DHCP 会自动根据客户端架构提供对应的引导文件：

BIOS PXE 启动
  • 引导文件: pxelinux/pxelinux.0
  • TFTP 地址: tftp://{{ server_ip }}/pxelinux/pxelinux.0
  • 配置文件: pxelinux/pxelinux.cfg/default

适用于传统 BIOS 系统，通过 TFTP 下载引导文件

UEFI 启动 (TFTP + HTTP)
  • x86_64 引导: grub/shimx64.efi (推荐)
  • x86_64 备用: grub/grubx64.efi
  • x86 引导: grub/shimia32.efi
  • ARM64 引导: grub/grubaa64.efi
  • TFTP 地址: tftp://{{ server_ip }}/grub/grub.cfg
  • HTTP 地址: https://{{ server_ip }}/boot/grub/grub.cfg

适用于 UEFI 系统，支持 TFTP 和 HTTP 两种方式下载引导文件

启动菜单同时支持 TFTP 和 HTTP 两种方式下载内核/initrd

DHCP 配置要求:

  • next-server {{ server_ip }};
  • BIOS: filename "pxelinux/pxelinux.0";
  • UEFI x86_64: filename "grub/shimx64.efi";
  • UEFI x86: filename "grub/shimia32.efi";
  • UEFI ARM64: filename "grub/grubaa64.efi";
